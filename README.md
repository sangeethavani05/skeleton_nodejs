# skeleton_nodejs

# Description

Middleware Skeleton with Express Framework

# Installation

```bash
$ npm install
```

# Running the app

```bash
# development
$ npm run start
```

# Test

```bash
# unit tests
$ npm run test
### This will run both eslint and Unit test with coverage

# eslint tests
$ npm run test:eslint
### Only for lining purpose

# API unit test with coverage
$ npm run test:jest
```
